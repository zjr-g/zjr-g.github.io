    这是关于MySql的学习记录

    使用的书籍是《Mysql必知必会》

    MySql的安装及学习环境的配置

    直接进入MySQL官方网站下载安装即可

    https://www.mysql.com/downloads/
    对应MySql版本为4.1及更高版本

    Linux下利用wget安装（版本可自行更替）

    wget https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.26-linux-glibc2.12-x86_64.tar.gz
    tar -xvf mysql-5.7.26-linux-glibc2.12-x86_64.tar.gz

    或者利用apt安装

    apt-get install mysql-server(mysql服务，必须安装)
    apt-get install mysql-client(mysql客户，非必须)

    可以自行安装图形化管理软件如：Navicat，MySQL Administrator。

    这里主要为了学习MySql的相关命令使用的是mysql-client命令行实用程序。

    练习数据库可以到如下网站下载（书籍配套练习数据库）

    https://forta.com/books/0672327120/
    在网页最下方下载mysql_scripts.zip即可
    解压后有两个文件：creat.sql，populate.sql

    连接数据库

    mysql -uxxx -pxxx -hxxx -Pxxx
    xxx为参数，-u后为用户名，-p后为密码，-h为MySQL服务器名或地址，-P为服务端口号
    本地连接则可以不带-h和-P
    注意参数格式

    连接MySql后

    create database crashcourse; （创建数据库）
    use crashcourse; （进入数据库当中）
    source create.sql; （创建表,要填脚本的绝对路径）
    source populate.sql;（插入数据）

    当然执行这两条命里需要用户有相应的权限。
    出现Query OK, 1 row affected (0.00 sec)则表示相应的表创建成功，数据插入成功。

    SHOW 命令的使用

    show databases;(显示当前数据库)
    如果当前账户的权限不够，则不能将服务器上的所有数据库显示出来

    show tables;(显示数据库中的表)

    show columns from customers;(显示customers表里的字段)

    show grants;(显示当前用户的权限)

    show errors;(显示服务器错误消息)
    show warnings;(显示服务器警告消息)
    show status;(显示服务器状态信息)

    之后的学习都将在这个数据库与这些表中进行
